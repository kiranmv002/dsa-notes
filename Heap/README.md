Heap Notes

Heap is a special type of Complete Binary Tree
that follows the heap property.

This folder contains simple and easy-to-understand
notes on Heap data structure.

Topics Covered

• Heap Introduction – Basics of heap and heap property.

• Heap Operations – Insertion, deletion, and heapify process.

• Heap Implementation using Array – Array representation of heap.

• Min Heap vs Max Heap – Comparison of heap types.

• Heap Sort – Sorting algorithm based on heap.

• Time Complexity of Heap – Performance analysis of heap operations.

More heap topics will be added as learning continues.
