Searching Notes

Searching is used to find an element in a collection of data.
This repository is created to store simple and easy-to-understand
notes on searching algorithms.

Topics Covered

• Binary Search – Efficient search algorithm for sorted data.

• Linear Search – Basic searching technique that checks elements one by one.

• Linear vs Binary Search – Comparison of linear and binary search techniques.

• Searching Examples – Step-by-step examples of searching algorithms.

• Searching Time Complexity – Best, average, and worst case analysis.

• Searching Summary – Quick overview of searching concepts.

More searching algorithms will be added as learning continues.
