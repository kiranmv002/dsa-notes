Graph Summary

Graph is a non-linear data structure
used to represent relationships
between different objects.

It consists of vertices (nodes)
and edges (connections).

What We Learned

- Graph Introduction
- Types of Graphs
- Graph Representation (Adjacency Matrix & List)
- Graph Traversal (BFS and DFS)
- Shortest Path (Dijkstra Algorithm)
- Time Complexity of Graph Algorithms

Important Concepts

Vertices and Edges
- Vertices represent nodes
- Edges represent connections

Directed and Undirected Graph
- Directed graph has direction
- Undirected graph has no direction

Weighted Graph
- Edges have weights or costs

Graph Traversal
- BFS uses Queue
- DFS uses Stack or Recursion

Graph Representation
- Adjacency Matrix
- Adjacency List
