Queue Notes

This folder contains simple and easy-to-understand
notes on the Queue data structure.
Queue follows the First In, First Out (FIFO) principle.

Topics Covered

• Queue Introduction – Basics of queue and FIFO principle.

• Queue Operations – Enqueue, Dequeue, and Front operations.

• Queue Implementation using Array.

• Queue Implementation using Linked List.

• Applications of Queue.

• Queue Time Complexity – Time complexity of queue operations.

• Queue Summary – Overall revision of queue concepts.

More queue topics will be added as learning continues.
